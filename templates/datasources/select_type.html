Working with Database Queries

How to create and use database queries in Hermes

To create database queries as data sources, follow these steps:

  1. Create a Database Connection - First, create a reusable connection to your database.
  2. Create a Database Query Data Source - Then create a data source that uses the connection.
  3. Add Queries - Define the SQL queries you want to run on the database.
  4. Configure Integration - Set up how query results integrate with user profiles.

This separation allows you to reuse the same database connection across multiple data sources, while keeping each data source focused on specific queries and integration points.
